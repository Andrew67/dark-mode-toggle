# Dark Mode

![App logo](./app/src/main/res/mipmap-xhdpi/ic_launcher.png)

 - System-wide dark mode toggle for Android versions 4.1 through 9.0
 - Offers quick settings tile on Android 7.0 and above

 - Google Play (coming soon)
